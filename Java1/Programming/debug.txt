eclipse에서 디버깅 하는 방법

1. 라인 왼쪽 더블 클릭하여 breakpoint 설정 - 다시 더블 클릭하면 해제

2. 상단 벌레모양(debug)클릭하면 설정된 breakpoint 라인 까지만 실행

3. Step Over (F6) - breakpoint 부터 코드 한줄씩 실행

4. Step Into (F5) - 함수, 사용자 지정 datatype 등의 정의된 부분으로 이동

5. Step Return (F7) - Step Into 하여 이동하였을 때 다시 원래의 코드로 돌아갈때 사용

6. Resume (F8) - 다음 breakpoint 지점까지 이동 또는 다음 breakpoint가 없을 시 디버깅 종료

7. Terminate (Ctrl + F2) - 디버깅 종료
